# CrowdGuard 🛡️
**AI-Driven Crowd Management Simulation**

CrowdGuard is an interactive web-based simulation tool designed to analyze and manage crowd dynamics. You can design physical environments, manipulate crowd behavior, and monitor real-time telemetry to optimize safety.

## Features
- **Environment Toolkit**: Dynamically build custom venues with tools like Spawn Areas, Entry Gates, Exit Points, Walls, Squares, Shops, Stages, and Washrooms.
- **Agent Simulation**: Simulate human-like crowd behavior with adjustable speeds and panic inducement levels.
- **Real-time Telemetry Dashboard**: Monitor critical statistics including the number of Active Agents, Evacuated personnel, and dynamically calculated Stampede Risk levels.
- **Density Heatmaps**: Visualize crowded zones and potential bottlenecks intuitively.

## How to Run
Simply open `index.html` in a modern web browser. No complex server setup, dependencies, or backend required.

## Purpose
This project is built to explore crowd behavior for pre-event planning and emergency evacuation strategies, helping to identify bottlenecks, optimize flow, and minimize stampede risks in a controlled sandbox environment.
